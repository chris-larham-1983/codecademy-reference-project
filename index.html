CSS Selector Cheat Sheet

This project draws heavily on the Complete CSS Selector Reference found at w3schools.com. The examples have been modified and the descriptions slightly re-worded for clarity.

Complete CSS Selector Reference
CSS Selector CSS Example Example Explanation
.class .first Selects all elements whose class attribute is equal to 'first'.
.class1.class2 .first.second Selects all elements whose class attribute contains both 'first' and 'second'.
.class1 .class2 .first .second Selects those elements whose parent element has a class attribute equal to 'first' and whose own class attribute is equal to 'second'.
#id #lastname Selects the element whose id attribute is equal to 'lastname'.
* * Selects all elements.
element a Selects all <a> elements.
element.class a.codecademy Selects all <a> elements whose class attribute is equal to 'codecademy'.
element, element a, figcaption Selects all <a> elements and all <figcaption> elements.
element element figcaption a Selects all <a> elements that are inside <figcaption> elements.
element > element figcaption > a Selects all <a> elements whose parent is a <figcaption> element
element + element figcaption + a Selects the first <a> element that is placed immediately after a <figcaption> element.
element ~ element figcaption ~ a Selects all <a> elements that are preceded by a <figcaption> element with the same parent.
[attribute] [class] Selects all elements with a class attribute.
[attribute=value] [class=brushes] Selects all elements whose class attribute is equal to 'brushes'.
[attribute~=value] [class~=brushes] Selects all elements whose class attribute contains the word 'brushes'.
[attribute|=value] [class|=brushes] Selects all elements whose class attribute begins with 'brushes'. The value must be a whole word, either alone ('brushes') or followed by a hyphen ('brushes-and-stuff').
[attribute^=value] figcaption[class^="description"] Selects all <figcaption> elements whose class attribute starts with 'description'.
[attribute$=value] figcaption[id$="image"] Selects all <figcaption> elements whose id attribute ends with 'image'.
[attribute*=value] figcaption[class*="verbose"] Selects all <figcaption> elements whose class attribute contains the substring 'verbose'.
element:active p:active Selects and styles the active paragraph (it becomes active when you click on it). Used most commonly on <a> elements.
element::after div::after Inserts something after the content of each <div> element.
element::before div::before Inserts something before the content of each <div> element.
element:checked input:checked Selects every checked <input> element (applicable to radio buttons and checkboxes). Also used for <option> elements.
element:default input:default Selects all the default <input> elements.
element:disabled input:disabled Selects all the <input> elements that possess a 'disabled' attribute.
element:empty div:empty Selects all the <div> elements that have no children (including text nodes).
element:enabled input:enabled Selects all the enabled <input> elements (i.e. those <input> elements which do not have a 'disabled' attribute).
element:first-child h1:first-child Selects each <h1> element that is the first child of its parent.
element::first-letter h1::first-letter Selects the first letter of every <h1> element.
element::first-line h1::first-line Selects the first line of every <h1> element.
element:first-of-type div:first-of-type Selects each <div> element that is the first <div> element of its parent.
element:focus input:focus Selects the <input> element which has focus.
:fullscreen :fullscreen Apply CSS styling when the page is in full-screen mode.
element:hover p:hover Selects paragraphs on mouse over.
element:in-range input[type=number]:in-range Selects all the <input> elements that have a type attribute equal to 'number' and are in-range.
element:indeterminate input[type=checkbox]:indeterminate Selects all the <input> elements that have a type attribute equal to 'checkbox' and are in an indeterminate state.
element:invalid input[type=email]:invalid Selects all the <input> elements that have a type attribute equal to 'email' and contain an invalid value.
element:lang(language) p:lang(fr) Selects each <p> element that has a lang attribute equal to 'fr' (French).
element:last-child div:last-child Selects each <div> element that is the last child of its parent.
element:last-of-type div:last-of-type Selects each <div> element that is the last <div> element of its parent.
:link a:link Selects all unvisited links.
::marker ::marker Selects the markers of list items.
:not(selector) :not(div) Selects every element that is not a <div> element.
element:nth-child(n) div:nth-child(2) Selects each <div> element that is the second child of its parent.
element:nth-last-child(n) div:nth-last-child(2) Selects each <div> element that is the penultimate child of its parent.
element:nth-last-of-type(n) div:nth-last-of-type(2) Selects every <div> element that is the penultimate <div> element of its parent.
element:nth-of-type(n) div:nth-of-type(2) Selects each <div> element that is the second <div> element of its parent.
element:only-of-type div:only-of-type Selects each <div> element that is the only <div> element of its parent.
element:only-child div:only-child Selects each <div> element that is the only child of its parent.
element:optional input[type=text]:optional Selects every <input> element that does not have a 'required' attribute and whose type attribute is equal to 'text'.
element:out-of-range input[type=number]:out-of-range Selects every <input> element whose type attribute is equal to 'number' and whose value is outside the specified range.
element::placeholder input[type=text]::placeholder Selects every <input> element whose type attribute is equal to 'text' and that possesses the 'placeholder' attribute.
element:read-only input[type=text]:read-only Selects every <input> element whose type attribute is equal to 'text' and that possesses the 'readonly' attribute.
element:read-write input[type=text]:read-write Selects every <input> element whose type attribute is equal to 'text' and that does not possess the 'readonly' or 'disabled' attributes.
element:required input[type=text]:required Selects every <input> element whose type attribute is equal to 'text' and that possesses the 'required' attribute.
element:root html:root Selects the document's root element. In HTML, this is always the <html> element.
element::selection p::selection Selects the portion of <p> elements that is selected (highlighted) by a user.
element:target p:target Selects the <p> element that is the target of an internal page link when that link is clicked.
element:valid input[type=email]:valid Selects every <input> element whose type attribute is equal to 'email' and whose value validates according to the element's settings.
element:visited a:visited Selects all visited links.
